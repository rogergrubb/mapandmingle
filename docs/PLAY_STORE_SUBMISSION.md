## Data Safety Section

### Data Collection

**Personal Info**
- Name: Collected (required for profile)
- Email: Collected (required for account)
- Phone: Not collected

**Location**
- Approximate location: Collected (for nearby features)
- Precise location: Collected (optional, for pins)

**Photos and Videos**
- Photos: Collected (optional, for profile)

**Messages**
- In-app messages: Collected (for chat feature)

### Data Sharing
- No data shared with third parties for advertising
- Data may be shared for analytics (anonymized)

### Security Practices
- Data encrypted in transit (HTTPS)
- Data encrypted at rest
- You can request data deletion
